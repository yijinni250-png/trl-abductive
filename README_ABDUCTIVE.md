# TRL-Abductive
In this repository, we have added the abductive preference learning methods to the original trl repo.
```
trl-abductive/
├── trl/
│   ├── trainer/
│       ├── adpo_trainer.py/        # Abductive DPO Trainer
│       ├── adpo_config.py/         # Abductive DPO Config
│       ├── multidpo_trainer.py/    # MultiDPO Trainer (Abductive DPO + DPO)
│       ├── multidpo_config.py/        # MultiDPO Config (Abductive DPO + DPO)
│       ├── multidpop_trainer.py/        # MultiDPO-P Trainer (Abductive DPO-P + DPO-P)
│       └── multidpop_config.py/        # MultiDPO-P Config (Abductive DPO-P + DPO-P)
```
